## Getting Started

### Prerequisites

- NodeJs

- TypeScript

- Docker

### Installation

1. Install Prerequisites
2. run backend
   ```sh
   cd backend
   npm run watch
   npm run dev
   ```
3. Run FrontEnd locally.
   ```sh
   cd frontend
   npm run dev
   ```
